Chlorophyll-a Concentration Predictive Model

Data sourced from the Copernicus Marine Service and the NOAA's ERDDAP servers.

Trained on 8 day averaged data from 2004 to 2021 in the 18-24°N and -110--104°W area.

Daily predictive map stored in output/latest_forecast_map.png, updated daily at 5 AM PST.

Data explorer tool at https://rysohn.github.io/chlora_model/.
